# Revikoo — Project Instructions

## Descripción del proyecto

Revikoo es una marca española que ofrece soportes de metacrilato con tecnología NFC para ayudar a negocios locales a conseguir más reseñas.

El producto principal es un soporte físico de sobremesa. No son tarjetas de visita NFC y no utiliza códigos QR.

## Stack técnico

- HTML5.
- CSS vanilla.
- JavaScript vanilla.
- Sin React.
- Sin Vue.
- Sin Bootstrap.
- Sin Tailwind.
- Sin procesos de compilación.
- Sin backend.
- Hosting mediante GitHub Pages.
- Dominio final: https://revikoo.com

## Reglas de contenido

- No utilizar códigos QR.
- No representar tarjetas de visita como productos NFC.
- No inventar testimonios.
- No inventar clientes.
- No inventar estadísticas.
- No prometer resultados garantizados.
- No afirmar que las reseñas posicionan automáticamente al negocio.
- No utilizar logotipos de terceros sin autorización.
- El producto debe mostrarse como un soporte de metacrilato.

## Objetivo comercial

La web debe:

1. Explicar el producto en pocos segundos.
2. Mostrar cómo funciona.
3. Transmitir confianza.
4. Presentar el precio con claridad.
5. Generar solicitudes por WhatsApp o email.
6. Estar preparada para incorporar compra online en el futuro.

## Calidad técnica

- HTML semántico.
- CSS centralizado.
- JavaScript centralizado.
- Diseño responsive desde 360 px.
- Navegación accesible mediante teclado.
- Contraste suficiente.
- Imágenes optimizadas.
- Lazy loading por debajo del primer bloque.
- Sin errores en consola.
- Sin rutas rotas.
- Rutas compatibles con localhost, GitHub Pages y revikoo.com.
